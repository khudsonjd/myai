Health

∞
Anything
Any effort that benefits from continuity
Why
AI forgets. This fixes that.
Without MetaTemplate
Every session
starts from zero
  • Re-explain your situation every time
  • AI gives generic answers, not informed ones
  • Context builds up, then disappears
  • Switching tools means starting over
  • Only works while the chat window is open
With MetaTemplate
Every session
picks up exactly here
  • AI reads the files and gets to work
  • Context accumulates and stays accurate
  • Any AI tool, same files, same result
  • Session ends — files updated, nothing lost
  • Works across weeks, months, and versions
How
Four files. Any effort.
The file structure
📌
SESSION.md
Active efforts + immediate focus — loads every session
👤
CONTEXT.md
Static personal context — who you are, your setup
📐
project_plan.md
Stable reference — goals, decisions, constraints
⚡
status.md
Live state — priorities, what's active, what's next
↓
✦ AI reads → orients → gets to work
Getting started
1
Copy the starter files
Clone the repo. Copy SESSION.md, CONTEXT.md, project_plan.md, and status.md into your workspace folder.
2
Fill in what you know
Add your project name and a one-line description. Leave the rest blank — your AI will help you build it out.
3
Start working
Open a chat, share SESSION.md. The AI reads the files, confirms it's oriented, and you begin.
4
Files stay current
As you work, the AI updates status.md. Next session: same command, full context, zero re-explaining.
How — No File Access
Works without file access too.
Browser chat, enterprise tools, or any AI that can't read your files — same system, manual sync.
The paste-based workflow
📋
Before each session
Paste SESSION.md + status.md into chat — AI reads it as context
↓
💬
Work normally
AI has full context — answers informed, no re-explaining needed
↓
💾
Before closing
Ask: "Regenerate my updated files." Copy output → save locally
↓
Next session: paste again — full context restored
Supported environments
✓
Claude Code CLI
Full file read/write. AI updates files automatically. Best experience.
✓
VS Code + Copilot Agent
Full file access in workspace. Same auto-update workflow as CLI.
~
Claude.ai / ChatGPT browser
Paste files at start. Ask for regenerated files at end. Manual but functional.
~
M365 Copilot (enterprise)
No file creation. Paste-based workflow only. Keep files locally, sync manually each session.
